### EMPLOYEE MANAGEMENT

## Setup

- clone the repo with git clone <repo_url>
- Run `npm i` command to install dependencies
- create a `.env` file and put the below ENVs in that file with appropriate values
  - PORT=8000
  - MONGO_URI=
  - DB_NAME=employee
  - ADMIN_PASSWORD=
  - JWT_SECRET=

## Run application

- Run `npm start` on the terminal
